int year(const int &yyyymmdd)

Returns integer -1 when year in yyyymmdd is less then 1000 or greater then 9999, or year in date representing with yyyymmdd variable. Variable yyyymmdd is an integer reprezenting date in form of yyyymmdd.

Example:

Date_interval d; //object d
if(d.year(20160912) != -1)
    cout << d.year(20160912) << "th" << endl; // 2016th
else
    cout << "Wrong date!" << endl;
