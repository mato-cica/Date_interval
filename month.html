int month(const int &yyyymmdd)

Returns integer -1 when month in yyyymmdd is less then 1 or greater then 12, or month in date represented with yyyymmdd variable. Variable yyyymmdd is an integer representing date in form of yyyyddmm.

Example:

Date_interval d; //object d
if(d.month(20160912) != -1)
    cout << d.month(20160912) << ". month" << endl; // 9. month
else
    cout << "Wrong date!" << endl;
