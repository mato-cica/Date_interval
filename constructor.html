Date_interval(const int &dbegin, const int &dend)

The Constructor. Instantiates object, initializes member variables and calculates interval in days between dbegin and dend. Variables dbegin and dend are integers representing dates in form of yyyymmdd.

Example:

Date_interval d(20150101, 20210101); //object d
cout << d.get_Interval() << " days." << endl; // 2192 days.
